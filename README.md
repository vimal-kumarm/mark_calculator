# 🎓 Student Marks Calculator

A beginner-friendly web application built using **HTML**, **CSS**, and **JavaScript** as part of my Python Full Stack Developer journey (Day 5 Project).

---

## 🚀 Live Demo
> Open `index.html` in your browser — no installation needed!

---

## 📸 Features

- ✅ Enter student name and marks for 3 subjects
- ✅ Calculates average marks automatically
- ✅ Assigns grade based on average (A+, A, B, C, Fail)
- ✅ Shows personalized performance feedback message
- ✅ Color-coded result (green for pass, red for fail)
- ✅ Input validation — alerts if fields are empty
- ✅ Clean and responsive UI

---

## 🧮 Grade System

| Average Marks | Grade |
|---------------|-------|
| 90 and above  | A+    |
| 75 – 89       | A     |
| 60 – 74       | B     |
| 50 – 59       | C     |
| Below 50      | Fail  |
